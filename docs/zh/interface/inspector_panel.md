# 检查器面板

**位置:** *Properties Editor (属性编辑器) > Takes 选项卡 > Inspector (检查器)*

Inspector Panel (检查器面板) 会按对象提供一个极其详细的观察列表 (watchlist)，显示出到底哪些物体被并入了级联系统，以及它们目前被指派的动作 (actions)、插槽及其子级数据 (材质、节点树、形态键 - shape keys)。

## 受管与锁定 (Managed vs. Pinned)

被加入观察列表的物体一般处于以下两种状态之一：

受管 (Managed)
:   主动跟随当前级联系统的分配规则。物体的动作由当前激活版本的 View Layer 级联逻辑自动下发且指派。

锁定 (Pinned)
:   无视并免除级联逻辑。无论你如何随心所欲切换 View Layers，被锁定的对象都将死咬并维持其当前自己的某个固定动作不放。通过点击某一行上的 **图钉 (pin icon)** 即可对其锁定。

!!! warning "解锁 (Unpinning)"
    尝试去解锁一个拥有自己特有固定动作的对象将触发一个确认对话框跳出。
    这是因为一旦解锁，它当前的自定义固定动作就会被立即移除并被级联的共享默认配置所覆盖代替。

## 观察列表 (Watchlist)

观察列表为你展示了当前 View Layer 里面所有拥有变动的相关物体：

| 专栏 | 描述 |
|--------|-------------|
| **Pin (图钉)** | 切换对象处于受管/锁定状态。 |
| **Name (名称)** | 对象名称（直接点击以在视口中框选它）。 |
| **Sub-data icons (子数据图标)** | 对应每个不同类型动画数据块（如同材质、形态等）的迷你快捷指示图标。 |

### 紧凑模式

在紧凑模式下，各个子数据图标会退化成一排极具辨识度的迷你标志挂靠在物体边上：

- **高亮图标** — 该数据块当前已经被分配指派了某个动作（动画）
- **黯淡图标** — 该数据块无有效动作或动画

单击这些迷你子数据图标任意一员，即可立即呼出用于直接管理各个数据块的悬浮操作框。

## 过滤器

你可以在列表顶部使用按钮过滤筛选你的观察表单：

- **Pinned 过滤** — 仅显示遭锁定的（被拔除体系）对象
- **Unpinned 过滤** — 仅显示当前受级联自动管理的对象

如果其下毫无对应结果的对象，这些过滤器图标将被禁用显灰 (grayed out)。

## 动作与插槽 (Actions & Slots)

选择位于观察列表中的具体某对象，并在面板偏下方深入挖掘它的详细信息：

- **Actions 列表** — 列出所有跟此物体挂钩或者曾经调用的动画动作 (Actions)
- **Slots 列表** — 显现处于当前动作下的所有动画插槽
- **内联重命名** — 在任何名称处轻轻双击，立即实地重命名修改它
