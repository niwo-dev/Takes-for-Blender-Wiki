# 导航面板

**位置:** *Properties Editor (属性编辑器) > Takes 选项卡*

导航面板是你管理整个 Takes 层级的主要控制中心。它包含 Takes 树状图、级联覆盖图标以及访问批量渲染的入口。

## 顶部控件

面板顶部会显示当前插件版本，并提供以下快捷按钮：

| 控件 | 描述 |
|---------|-------------|
| **Lock (锁定)** | 锁定当前面板以防止意外的状态更改。 |
| **Autokey (自动关键帧)** | 同时开启/关闭所有场景的 Blender 自动关键帧功能。 |
| **Timeline Sync (时间线同步)** | 跨场景同步当前的播放头位置。 |
| **Warnings (警告区)** | 显示当前的活动警告信息（比如预设丢失、悬空丢失的动作）。 |
| **Globals (全局设)** | 打开全局的级联设置。 |
| **Help (帮助)** | 打开文档（即此 Wiki 网页）。 |
| **Settings (设置)** | 树状图可见性及界面设计选项。 |

## Takes 树状图

Takes Tree (树状图) 是一个统一的分层列表，可总览你的整个项目结构：

```
📁 Scene Group (场景组)
  🎬 Scene (场景)
    📂 VL Group (VL 组)
      🔲 View Layer (视图层)
        📌 VL Version (VL 版本)
```

### 树状图行元素

每个 View Layer 行会显示多个级联覆盖图标 (cascade override icons)。通过这些图标你能一目了然地看出当前哪一层级别上有属性被覆盖了：

- **Ghost (幽灵)** — 静止状态 (Rest State)
- **Tag (标签)** — 指派的颜色标签
- **Variant (变体)** — 变体切换的激活状态
- **Action (动画)** — 级联动画动作参数
- **Compositor (合成器)** — 节点树覆盖
- **World (环境)** — 世界/环境背景覆盖
- **Camera (相机)** — 激活相机指派
- **Output (输出)** — 输出标签/规则
- **Render (渲染)** — 渲染配置预设指派

!!! tip "级联图标的折叠显示"
    当面板宽度较窄时，由于空间有限，级联图标会自动折叠收纳进一个
    带有 **⋯** 符号的按钮中。点击它即可打开一个悬浮窗口以完整查看所有图标。

### 树状图连线

可配置的缩进连线用于在视觉上呈现这个层级结构。连线可以继承所指派标签上的颜色标签，便于快速识别。

## 设置弹窗

点击 **齿轮图标** 访问树状图的显示设置：

| 设置项 | 描述 |
|---------|-------------|
| **Show Tag Color (显示标签色)** | 根据指定的标签色对树状缩进线进行染色。 |
| **Line Width (连线宽度)** | 细线 / 中线 / 粗线。 |
| **Icon Visibility (图标可见性)** | 逐个开关具体级联图标按钮的可见性。 |
| **Preview Size (预览尺寸)** | 行内 VL 实时预览微缩图的尺寸大小 (24/32/40 px)。 |

## 警告提示

一旦检测到问题，导航面板将会发出警告提醒：

- **Missing Preset (预设缺失)** — 这是因为级联指派了一个已被删除或找不到的 JSON 预设文件。
- **Dangling Action (悬空动作)** — 一个动画数据即将彻底丢失，这是因为你可能暂时禁用了针对该对象动画层级挂接的保护措施。
