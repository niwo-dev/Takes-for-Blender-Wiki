# 工作流：动画变种 (Animation Variants)

此工作流展示了如何在每一个独立的 View Layer 上，天衣无缝地设定和安排截然不同的动画剧本动作 —— 你比如说，同一个产品在一个 View Layer (视图层) 里执行着丝滑的 360 度圆盘自旋，在另一个层里玩着 45 度侧倾，而到了第三个层里却直接炸开成爆炸分解图。而且它们每个层还都各自享有独立的定点相机机位搭配！

## 布置片场起点

### 1. 创立专司各职的视图层 (View Layers)

给你每一条动画表演预先开个 View Layer 作为演出地盘：

1. 展开大 Takes Tree (Takes 视图树)。
2. 为每一个剧本单独按一下 **+** → **Add View Layer（添加视图层）** 按钮。
3. 请大发慈悲为它们做个明了直白的起名："Spin (旋转)", "Tilt (摇摆倾倒)", "Explode (炸翻天)"。

### 2. 将此等编纳一气 (Group Them)

1. 选择带头那个 VL 视图层。
2. 双指同敲 ++ctrl+g++ 为其打个包，建立并重命名叫 "Animations（我的大动画）"的组。
3. 把另外那几个 VL 全都乖乖鼠标拖进去作伴。

### 3. 给各片场委派专门的相机机位 (Assign Cameras)

既然它们动画都大相径庭了，也理应匹配自己独门的取景镜头角度才是硬道理：

1. 在你需要设定的那行 VL 上找到且点击那个相机小图标。
2. 点进去从中拣选一台恰当对应的摄影机出来任职。

## 正式开机布设动画 (Animating)

### 4. 层级动画私享制 (Animate Per View Layer)

1. 点击以切换进入那个叫 "Spin (旋转)" 的独门 View Layer。
2. 我们的大管家引擎 Cascade（级联）此刻会在底下神不知鬼不觉为当前这层 VL 分出专线独立的动画行动档 (action) 备着了！
3. 开心去操作摆弄添加各类动画节点。你捏在手心打上的任意关键帧统统都会单独仅记在这个 VL 独家的行动档内！
4. 那现在你切回并跳到那个 "Tilt (摇摆倾倒)" 视图层看看 —— 所有刚刚被你施过法的物体都会极其听话的被瞬间硬打散并拽回到原点中立默认姿态 (Rest State 基准状态层) 下！
5. 在这就不用客气，另外直接生造一套完完全全和刚才没有牵连的动画。

!!! tip "神奇魔术之手: Rest State (静止态)"
    全靠强大的 Rest State 体系时刻不离在紧逼监督看管。它才敢打保票
    就在你跳出那个挂有自己动画的 VL 一刹那间，强制勒令所有表演对象都变回起初默认原点身位
    。而且全部都是纯纯自动化代劳的。

### 5. 走起! 打包量产渲染 (Render All Animations)

1. 去左侧栏按亮多选系统 (multi-select) 开启那个多项操作的大选模式。
2. 将这好些个有参演动画戏份的 View Layers 的空全给勾满锁住选好。
3. 捅开那个 Render (渲染出图) 总出闸按钮。
4. 咱们那个极灵性的 U 形大出口 Smart Output (智能归档库) 将用他们各人所属之 VL 文件名，规规矩矩的帮你整理并且印成独立片名档案吐给你存盘！

## 最终成效总结 (Result)

如今每一个独立隔离的 View Layer 都有着它这辈子能独享的：

- 只它才有用的特配专属相机及取景角度！
- 绝不串台也互不干涉的特权剧本动画记录档！
- 绝不会发生覆盖覆写事故的专属导出成片美名档！
- 然后以上一切的指挥，仅靠那一颗统一调度的树状控制杆，一手把玩！
