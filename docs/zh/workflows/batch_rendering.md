# 工作流：批量渲染 (Batch Rendering)

这篇教程会深入详说怎样设定和开动一轮横扫全盘的完整多视图层 (View Layer) 级大出图队列任务。它一揽子全托附带我们级联设定及智能输出 (Smart Output)。

## 前置资格 (Prerequisites)

- 打开已建立了一打 View Layers 层配置齐聚的场子。
- 级联参数已做各家指派下发 (各挂各相机，各用各世界气氛乃至定好规格画质)。
- 你把 Smart Output 怎么整理的图案路径的字符排版全定死打妥。

## 配置智能出图输出路数

### 1. 定调输出规格格式 (Define Output Pattern)

1. 转进属性大厅区 **Properties > Output**。
2. 解锁激活 **Smart Output** 神系统。
3. 往目录编排串码填这个 (Directory Pattern):

    ```
    //renders/[scene]/[view_layer]/
    ```

4. 往底片名称排列法里灌入 (File Name Pattern):

    ```
    [view_layer]_[camera]_####.[file_format]
    ```

### 2. 派放参数规格套包 (Assign Render Presets)

为了能够掌控全维度一致而毫无参差的大画质:

1. 先在原厂面板随性定下最对你胃口的各种出胶品相 (啥级别显卡，多高抽样精度乃至 4K分辨率啥的)。
2. 把那些都调顺心遂意后，开我们那个小齿轮窗将其整个收封印为 Render preset 存盘备下神兵。
3. 接着把它直接抛向挂顶最上源的 **Global(全宇宙全局向)**，强行逼其全数下辖 VL 乖乖通吃此一最高标准套版，有钱任性！如果个别要搞差别那就仅挂去部分大组里！

## 鸣枪！起送批量车队！

### 置于光天化日跑 (Foreground Mode)

1. 点击进入 Takes Tree 专门的那条侧开列表区内里来。
2. 击穿那发引爆器 **Render(相片)** 牌子钮。
3. 就在这秒, 一整串序列都规规矩矩照这名单由小弟由上而下串烧往下跑：
    - 把场景舞台 / VL大幕一次次切位。
    - 各吃自己嘴内那被规定拿下的相机/天空光照和产品皮包变体色盘（级联套件）强行穿上伪装。
    - 收受这等级独享有指派画质采样书（预置档）。
    - 开全马力生成算图直接甩给老管家 Smart Output 去存档交接！
4. 任何当期实况演进之状态都在我们专造的读图表中进度化供呈！

### 扔回无边深黑底子里摸跑 (Background Mode)

1. 按这个形如上大显示器的按钮代替图片键 **Desktop (桌面级)**  就中了！
2. 你的本体程序 UI 操作体验不会因为他在后台跑大作业而生卡断腿慢如蜗牛, 尽管这同时他在底层暗影位中连拉出了各开路算子偷偷跑活!
3. 这些各房树上挂着的迷你监控器（那一个个视图层小照片/thumbnails）自会一个接一个排着点亮刷新成最终画质宣告完成落成出锅。

## 实地进度监测系统 (Monitoring Progress)

边廊上附赠的实时进度面板有在报告：

- 会为下阶每一列小将跟进打出完成进度表线百分数。
- 加批注并印上当道所过情况大字报 (等待接客Pending，引擎转热Rendering，大笔落款书写硬盘Saving，功成回朝Done...) 
- 要真死半路翻车的列车则会悬一大大 **红灯牌(Failed)** - 你只要用那光电指向红区便可浮现那死因为何症的诊断验尸报告！

## 清算出图大盘收宫之态

假如你按我在一开始瞎填教的那堆排字板（patterns）所走，那你打开硬盘那刻终有如下归整干净如艺术级神清气爽文件大集合展现:

## 暗送点内功建议 (Tips)

!!! tip "打版报捷打赏声 (Completion Sound)"
    只要阁下去开一趟到 **Preferences > Features** 里加插了叮当之音后。之后你就算跑出去做贼玩，也能被这个一旦 Background (黑户隐跑)收监大成全完工下线之刻敲你的那响巨锣（抑或自制`.wav`定制）而唤醒接单验货！

!!! tip "试着过目再定版 (Preview Before Render)"
    这绝技只拿两套组合动作即可： ++alt++ 加左按点击敲下任何那位列班长身前的渲染发启动画按钮，就瞬间回拨调回显示上一次其已经流出来的定相老照片了！根本就不必动干戈浪费一次跑马电费再重新强做图算一水。

!!! warning "天塌前必须强打存档印! (Save First)"
    若你想走那种完全分身隐形的黑打工 (Background render) 手位。先存妥交大作业 `.blend` 实身真盘案!此为极严律准！至于正面上战场的白干 (Foreground) 人呢？没事，因为正宫白手操作本身即生处于当前现活的缓存里吃记忆态(memory state)。
