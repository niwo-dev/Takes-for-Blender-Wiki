# 级联系统 (Cascade System)

**Cascade (级联系统)** 是 Takes for Blender 的超级核心计算引擎。它具有恐怖的多达 6 个层级的数据计算渗透阶梯决议能力，它赋予了你在上方任何层级都可以随意使用属性数值覆盖（override) 或者阻挠来自底下任何数值流的能力。

## 工作原理

就在你随便点击切换到任何一个 View Layer (视图层) 时，系统级联会去挨个访问当前需要的各类核心数据属性值 (譬如要用什么相机、哪个渲染背景、该不该用合成器预设档等等)。它的计算流程是由顶向下式过滤下渗 (**top-down**)，而且沿途只要找到它路过的 **首个非空指令数据**，那么它就会采纳该值并终止往下查：

```mermaid
graph LR
    G[Global/全域] --> SG[Scene Group/场景大组]
    SG --> S[Scene/细分场景]
    S --> VLG[VL Group/VL群组]
    VLG --> VL[View Layer/独立视图层]
    VL --> VLV[VL Version/最终底层版本号]
    
    style VLV fill:#e87d0d,color:#fff
```

## 层属性覆盖权限 (Override Tiers)

| 层级名称 | 控制辖区 | 举例场景 |
|------|-------|-------------|
| **Global(全域/全局)** | 管控囊括项目下所有层、群、景 | 全工程项目默认相机、主环境氛围灯光 |
| **Scene Group** | 其同名组内的所有所含次级场景 | 整个别墅群通用的同一套室外 HDRI 打光 |
| **Scene** | 单个场景内的全数 VL 子集成员 | 这个房间专门用的光散射合成器特效 |
| **VL Group** | 小组里面所有的成员 | 大家统统看向同样的一枚摄像机不要乱掉! |
| **View Layer** | 仅且唯一的一张视图层 | 每个独有的定制过场、动画以及动作背景 |
| **VL Version** | 特定存档快照包 | v2 版本特意给某处打点小细节微调 |

## 受理级联规则过滤的属性 (Cascade Properties)

目前能够完全被上述漏斗式计算截获支持的数据包含了以下种类：

| 属性内容 | 诠释概要 |
|----------|-------------|
| **Camera (相机)** | 决定究竟由什么确切的对象相机实体来掌镜最后的影像产出。 |
| **World (大气环境)** | 谁会被选作为提供本层全量全局光照环境反射。 |
| **Compositor(后期合成器)** | 用哪个基于后期节点连接处理网络。 |
| **Action (特许动画)** | 目前要让哪个主角动画附身跑帧做运动？ |
| **Render Preset(渲染规则/包)** | 有关渲染核心采样的 JSON 数据配置组合件。 |
| **Camera Preset(相机光学包)** | 控制摄像设备虚化或各项数据的 JSON 件设定。 |
| **World Preset (环境节点套)** | 色温, HDRI贴图配置档封装件。 |
| **Output Rule (导出演算规则)** | 走标签式动态智能输出路径算法体系。 |
| **Camera Rule (选景机位规则)** | 走自动选取指定标签之机位的自动化规则引擎。 |
| **World Rule (打光选择规则)** | 标签化自动寻找挑选出配套的光影规则体系。 |

## 执行设置参数下发 (Setting Overrides)

### 利用可视化的级联专属接孔 (Via Cascade Icons)

你只要拿鼠标签轻点树干末端对应的任何类别对应的小缩略图标即刻就会触发其菜单选项控制窗口(popover)。随便塞个什么选择给它，你就直接给此层安上了个无法穿越覆盖的“铁令霸权锁”。而点击面板内的“叉/清楚”则解除你的指令放其让数值继承权限流至上一级（父母层）掌权。

### 依据统一侧边汇总窗 (Via Context Properties)

上下文设定汇总面板 (Context Properties)，也就是左边的这大块长版，可以统统把现在身处这特定激活的 VL 层身上披挂了啥所有的特权、限制参数以及它继承的乱七八糟血统的各种指示器，给一览无遗罗列整齐在一块集中给你做检阅。

## 视觉颜色暗示 (Visual Indicators)

- **明亮的靓丽色彩图标** — 很确切，这就是这里当地土生土长的独立硬指标
- **黯淡的透明发灰图标** — 非常乖巧，它是随波逐流拿来自头上老爸老妈留下来的恩宠（继承）
- **按住 Alt+右键 (Alt+Click)** — 最残忍也最有效的一键洗白该级内所有硬写入指标回归其天然的传承态！

!!! tip "开发者查漏神器 (Cascade Debugging)"
    用鼠标轻轻掠过 (Hover) 图标且停顿一小秒钟即可弹出解释：
    你可以立马看透此时现在这里运用的参数变量底裤——它是最终上溯到了哪一级
    谁赋予给它的特权，一目了然抓错误查臭虫好手。
