# 智能无脑导出管家 (Smart Output)

所谓的 **Smart Output (灵性自动导出系)** 为每一次需要导出、渲存的操作动作统共上供并搭载了强无敌的全动能基于各种动态神算令牌解析替换（token-based file path resolution）法则去完美规划去向结构体系！你不再会面临枯燥到每做一次渲染输出便得累巴巴人工手动去拼写每张死板的档案文件照！取而代之地是被无尽的高智系统采用一切围绕当时环境及背景信息自生成与取代组合拼构之字串参数 (tokens) 去帮您打发命名活！

## 怎么使唤这个强大的管家系统 (Enabling Smart Output)

1. 先去请出咱 Blender 自己内带属性版面上头那个很重要的总开关项区域板 **Output (输出主项)** 里候着。
2. 很轻易在上面找到那个闪亮的唤醒复选框并击打勾选激活点亮系统： **Smart Output(自动化无脑分拣包)**。
3. 着手搭建一个能让系统帮自己建好子分库大目录文件夹的 **Directory Pattern(分类档阵列符串码)** ；连带着给那套给将来一切产子取好字号名字排好的字签 **File Name Pattern(命名法列串码)**! 此环节须动用参数标记器令牌法 (tokens)。

## Token 解析与编写的咒法 (Token Syntax)

一切代表了高深运算变量值的神之变量字符令牌必定要用那特供的一套可由你们自定义配发的特制挂号括弧系统包裹才起效识读。当前原厂初始给您配置的便是极为普通、标准也最大众认知的 —— [方脑壳包裹法(square brackets)]:

```
[view_layer]_[camera]_####.[file_format]
```

这就这麽一段无聊简短的神符丢过去让系统翻译解算，最终必定化解落地被输出还原生成类似于如此有内容而且详实饱满无误的成果真名: `Front_3-4_CamHero_0001.png`

### 各流派各风格支持可选的封装标记 (Bracket Styles)

你可以去插件老窝最深处做调整挑选属于你最舒服的使用法: 一头扎进 **Addon Preferences(偏好全集) > Behavior Options (全家行为习性大选)> Syntax Brackets(句法挂括选取框)**:

| 挂符做派体 | 长着副什么鬼德行(Syntax) | 如果套用的示范结果样版 |
|-------|--------|---------|
| Square(原汁原味小方块法) | `[神之令牌名称]` | `[view_layer]_####` |
| Curly(卷毛大波浪包裹) | `{神之令牌名称}` | `{view_layer}_####` |
| Angle(小锐角箭头标枪) | `<神之令牌名称>` | `<view_layer>_####` |
| Parens(柔滑平圆包法) | `(神之令牌名称)` | `(view_layer)_####` |
| Percent(吃满全点满的分成法) | `%神之令牌名称%` | `%view_layer%_####` |
| Dollar(就是浑身散发铜臭味套) | `$神之令牌名称$` | `$view_layer$_####` |
| Hash(超高冷尖锐切割包) | `#神之令牌名称#` | `#view_layer#_####` |

## 百般齐全随时任遣调的词句真源字符 (Available Tokens)

### 与所身处之环境场息息挂牵系的一类 (Context Tokens)

| 此令牌打上何码 | 它将转化显形显化何真字 | 生成的实操后形如 |
|-------|-------------|---------|
| `scene` | 当下在发功主导的场景正称 | `Kitchen` |
| `view_layer` | 你现所处的 View Layer 名讳 | `Front_3-4` |
| `camera` | 目前场内打头阵相机代号名 | `CamHero` |
| `group` | 当下的 VL Group 组别头衔 | `Hero_Shots` |

### 牵动各类动画轨迹时间线脉络法号 (Animation Tokens)

| 此令牌打上何码 | 它将转化显形显化何真字 | 生成的实操后形如 |
|-------|-------------|---------|
| `frame` | 正处于游标点上的关键时间指针计数 | `42` |
| `frame_start` | 整片剧集的出闸时间记录号点 | `1` |
| `frame_end` | 半个最高末尾段结的打版点 | `250` |
| `frame_range` | 从头到尾覆盖区间牌 | `1-250` |
| `action` | 本次场景当前激活原动作名！ | `CameraFly` |
| `tks_action` | 最正宗受整个级联体系敲定的动画名号 | `SpinX` |

### 紧扣发功产出图片规格参数类印号 (Render Tokens)

| 此令牌打上何码 | 它将转化显形显化何真字 | 生成的实操后形如 |
|-------|-------------|---------|
| `engine` | 究竟是哪家出苦力计算法引擎操盘 | `cycles` |
| `samples` | 出单渲染时抽多少数量样本密网去打底 | `128` |
| `res_x` | X 轴输出分辨率界限 | `1920` |
| `res_y` | Y 轴输出分辨率极限 | `1080` |
| `file_format` | 具体什么出图性质格式图表后缀 | `png` |

### 吸取全底层操作运转法则环境内参 (System Tokens)

| 此令牌打上何码 | 它将转化显形显化何真字 | 生成的实操后形如 |
|-------|-------------|---------|
| `blend` | 这个系统包最初存世大归档源名 | `project` |
| `date` | 在这片时空中诞生此刻打下刻出来的历法纪年 | `2026-03-29` |
| `time` | 就在这发力瞬间钟表摆去几点记录 | `14-30-00` |
| `sep` | 专职分隔用作断界切的号位符号体 | `_` |

!!! tip "不学不背之超级作弊小神板 (Cheat Sheet)"
    你记不住这么多头串代称码不要紧！去面板的抬头边找一个 **小小薄薄的一本书册图样 (Syntax Tokens)**! 
    只需点一点！便能下拉开启天罗地网分类且能即生词块拷贝参照的大字典！全供即兴抄写享用。

## 全球断字离符法则与间隔绝法 (Separators)

咱们能最多容下提供足量上达 5 枚互不相干，完全特立独行的系统隔断法则令牌（分别是 `[sep1]` 直至 `[sep5]` ）: 你凭心安置使用这些去当作隔膜分隔层好用来分化：

- `[sep1]` 被设置做 `_` 完全可作为隔绝档和打分类来使用
- `[sep2]` 当做 `-` 可以连带同类别或者变量作衔接
- `[sep3]` 取 `.` 专门去包囊那一小撮琐碎修订代号用以截取档。

诸位快移步请到总心 **Addon Preferences (总系统偏好及界面选)** 里随你们爱好选上你们特爱地专属绝断截断符了呗。

## 图片排期编号和数字阵 (Frame Numbers)

只要照套运用 Blender 圈内皆知的通用补齐站位格式口诀法号 `####` !咱们引擎就懂得乖乖帮你们配好带齐 (0001, 0002...照此类排) 的四位填零位大序列长龙。当然，你写在这里只要一落笔，实时预览便马上用大字给你的最终真名成果给你即写即显挂个样了呢！
