# 变体切换 (Variant Switch)

**Variant Switch** 是专门针对多规格产品的强大变种控制体系 —— 能实现同一物品在不同的多材质、甚至在不同的颜色款式方案下自由做更换。配合上针对特约组别的无痛材料调包（material swapping），同时天然继承我们独门秘方 "极度关联下级联感知系统 (cascade-aware)"。 

## 使用理念

但在做任何商业类型展现以及进行大规模的产品效果表现时，客户的需求往往如出一辙而枯燥重叠：他们要求对他们一款型号相同的比如手表产品，渲染输出 "奢华土豪金", "太空冷银", 或者所谓的高级"磨砂熏黑" 。这就要求你需要不计其数的去手工切换、更替。我们的 Switch 正是用于彻底自动化将一切此类无聊材质换装的工作完美闭环包办！

## 等级结构划分 (Hierarchy)

| 构成成员 | 工作释义 |
|---------|-------------|
| **Product (总代/产品大项)** | 最顶层最大的囊括整体。 (举例：总项目 "天梭腕表", "农夫山泉水瓶" 等)。 |
| **State (衍生状态 / 变体款型)** | 已命名好的特定变种。(举例: 在手表里面有个名为 "闪光金 (Gold)"的特定全配置档案款 )。轻按即立刻观赏变化。 |
| **Part (零部件块)** | 拆零散的具有实体的组件(而且死死贴在具体几何体或者集合模型上)。 (举例:这部分属于 "表身", "那部分归为水杯上的背带")。 |
| **Pool (储备水池/库存栈)** | 只效力给各自所属零件池内的独享且无穷尽的可调用原材料槽。 |

## 操作流程指南

### 1 创建您的产品系

1. 在侧边面板区域的选项中切到专司业务的 **Variant Switch** 面板上去。
2. 用力戳中间的加号大钮 **+** , 不然就是老式快快捷方式 ++ctrl+n++ 也是能立刻建立一条叫做 Product (产品总线)的项目的。
3. 一旦建好了最顶层, 很懂事的插件默认会免费赠送附带建立了个基础散碎模块叫 ("base") ,然后带了一个准备好的初始库存位虚席以待。

### 2 加入花色款式 (Adding States)

1. 先在上面选中这个 Product 层框框以宣告你的工作范围确立。
2. 无脑加一条下阶衍生状态 (State) 并为其定调 (比方起名为: "闪光纯金 / Gold" )。
3. 这些所有叫衍生状态（State）的东西其实自己并不负责拥有啥模型实物的贴图表现，它只不过是个发号施令的指针系统! 它只保管记忆一串索引代号 (pool index)! 用下指令的方法指示每一种零部件应调用材料池底下位于几号牌子箱子里的装备！

### 3 给系统注入物料补给 (Assigning Materials)

1. 选择一个零部件模块 Part 把它的小箭头下拉出来看看里面空荡的装备库存 (pool)。
2. 把物料赋予空空如也拿编号1打头位置 — 然后好戏上演, 系统背后自动生成一排带编码2号空箱子接盘。
3. 当然这是靠调取了 Blender 本自建且带记忆机制和全知视角的材质选择下拉控制中心功能！

### 4 秒切换欣赏您的伟大造物

想要切就只需要轻踩那几个没有点亮的灰暗的 **钻石标志 (diamond icon)** 任意想展示的状态字眼头上 , 大银幕上实时同步更新。当然哪一款真正雄霸当前被翻拍挂起着的时候身上顶的必定是一颗被实体填塞漆满的黑色大原点标志。

## 工作切换流模式 (Variant Modes)

我们 Switch 开关提供涵盖一切需求的三大套不同的干预机制：

=== "Swap (简单生切硬换模式)"
    它极其无脑也最为耿直。 材质选项 A 全面对材质选项 B 进行暴力取代。
    
=== "Preset (深层预设修改法)"
    不把原有的本源材质废掉但是从外部通过高深的 JSON 强改里面一切能见得到的设置点、物理反馈等等属性（比如变透明或者变粗糙）从而直接更改面貌!

=== "Pool (庞大水池挑选体系)"
    这是一个宏观级别控制全产品生态链的体系法宝。每一款零碎的 Part 部件, 自己有个大仓库存（池子）, 每个叫做 State 的头目, 他只是发布了所有部件都得在对应当前指令下标（比如2），把对应的2号装备领出拿穿上的动作法则!

## 联动绝招与级联体系结亲

通过上述大纲，所有设定和编排过得Variant Switch 系统规则统统成为了被编录进 Cascade (级联系统)法典内极其合法与标准属性覆盖动作其中一环部分而已。你大可非常变态得告诉机器 — 第一台摄像机所照 View Layer A 必须要绑给我套用款式一 (State A) 的表面漆; 第二台摄像机 View Layer B 请让车表面漆呈款式三(State 3)！这完全合法合理。

!!! tip "标签大杀器 (Variant Tags)"
    尽速运用系统下 "Variant(产品组)" 类型之中的变色标签页分类赋予给各位 "States" !
    这些标签名将会完美成为通向 "Smart Output(智能无脑输出系统)" 最犀利的参数代词。你在出图填命名表时无脑打个代名词：`{variant_tag}` 变量。插件导出图的时候一切给你填平对齐分类!
