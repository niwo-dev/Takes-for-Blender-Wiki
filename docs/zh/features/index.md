# 核心功能 (Features)

Takes for Blender 提供了一套极具深度的工具包，专门用于管理复杂度极高的产品可视化渲染管线。每一项独立的功能也都被设计为可以经过系统级联（cascade system）完美互通协同工作。

| 功能 | 简介描述 |
|---------|-------------|
| [**Takes (镜头/大纲)**](takes.md) | 在拥有严谨层级关系下，将 View Layer 进行分组。 |
| [**Cascade System (级联系统)**](cascade.md) | 高达 6 个层级属性优先级覆盖的决议分配引擎。 |
| [**Variant Switch (变体切换)**](variant_switch.md) | 按层级、区块分配管控范围的产品变体生成与换料 (材料替换) 系统。 |
| [**Batch Render (批量渲染)**](batch_render.md) | 支持内置排队管理系统的全自动化全 View Layer 渲染队列引擎。 |
| [**Render Presets (渲染预设)**](render_presets.md) | 全基于 JSON 封装的轻量化预设档，统括渲染质量、相机、环境世界及导出设置。 |
| [**Rest State (静止基准状态)**](rest_state.md) | 全自动化侦测记录原初状态属性数据用以防丢/复原的关键帧镜像体系。 |
| [**Smart Output (智能输出)**](smart_output.md) | 利用动态令牌 (token) 变量进行文件存储路径与名称自动指派。 |
