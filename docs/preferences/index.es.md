# Preferencias

**Localización:** *Edit > Preferences > Add-ons > Takes for Blender*

O haz clic en el **ícono de engranaje** en el encabezado del Navigation Panel y selecciona **Open Preferences**.

## General

| Ajuste | Por defecto | Descripción |
|---------|---------|-------------|
| **Autosave Preferences** | Activado | Persistir los ajustes a un archivo JSON entre diferentes sesiones de Blender. |
| **Debug Logging** | Desactivado | Habilitar el registro de actividad por tema en archivos de registro diarios (logs). |

## Funciones (Features)

| Ajuste | Por defecto | Descripción |
|---------|---------|-------------|
| **Auto-Assign VL Actions** | Activado | Asigna automáticamente las acciones de cascada por cada View Layer local. |
| **Auto-Create Reference Action** | Activado | Espejea keyframes directamente hacia la Acción del Rest State (Reference State) automáticamente. |
| **Auto-Rename Actions** | Activado | Renombra acciones de forma estandarizada para coincidir con la convención de nombramiento en cascada generalizada de la arquitectura. |
| **Preset Automation Mode** | Manual | Cómo los eventos y variables por preset editables y sus opciones iniciales se tratan y operan de fondo directamente (auto-accept, sync, manual). |
| **Batch Render Sound** | Ninguno (None) | Define audio opcional interactivo o señal final confirmando que ha terminado una sub-tarea dependiente por lote unificador secundario (Render background parallel completion inicial). |

## Comportamiento (Behavior)

| Ajuste | Por defecto | Descripción |
|---------|---------|-------------|
| **Syntax Brackets** | Cuadrado `[]` | Patrón preestablecido e identificativo encriptando a nivel visual las variables por sintaxis directa para Smart Output y los tokens iniciales base. |
| **Separator Characters** | `sep1`=`_` | Hasta 5 parámetros oficiales limitados interactivos (tokens separators) individualmente modificables paralelos. |

## Informes Oficiales Secundarios Parciales Secuenciales Operativos Originales Centrales Identificables del Addon (Debug Logging de sistema nativo y registros base originarios genéricos e indexados por base paralela conectada al usuario inicial nativo local interactivo principal de fondo del archivo de programa base)

Cuando encendemos los modos interconectados generales de control preestablecidos operativos secundarios del debug para los logs primarios, los escritos formarán paralelamente su nivel de detalle oficial escribiéndolos cada día inicial nuevo y bajo apartados específicos:

| Tema (Topic) | Subtemas Operativos Originales Iniciales Secundarios |
|-------|-----------|
| **Core** | VL Switch, Cascade |
| **UI** | Inspector |
| **Ops** | Animation |
| **Features** | Reference State, Variant Switch |
| **Render** | Batch Render |

Botones o selectores nativos operativos generales de validación directa interactiva primaria paralela original a sistema base por nivel (`Debug`, `Info`, `Warning`, `Error`) limpian u descartan información particular antes a que un mensaje principal termine formateado oficialmente para almacenado secundario central operativamente hablando al texto directo unificable principal al disco y final. 

!!! note "Identificando la Ruta Final Operativa Interconectada Directa Oficial Originaria del Archivo Interno Preestablecido Inicial de Guardado (Log Location)"
    El guardado general y local principal origina un log base para cada archivo principal central con la matriz de nomenclatura `takes_for_blender_YYYY-MM-DD.log` internamente pre-posicionado alojado centralmente original base configurada interactiva pre-creado localmente generada secundaria originaria base paralela a directorio de logs (Log Directory / Add-on data central operativo interno de usuarios local nativo / Directorio general unificado original oficial de Takes local originario nativamente integrado). Los formatos u archivos diarios tras haber cumplido y originado localmente 7 jornadas originarias operativas temporales y temporizadores diarios internos son desechados solos con funciones internas automatizadas del originador de base paralela final.
