# Preview Render Preset

!!! note "Planned"
    On the roadmap, not started yet.

**Theme:** Feature · **Area:** Render

Reusable capture settings for view-layer preview thumbnails, so you don't reconfigure before each capture.

## Phases
- Define the preset (samples, denoise, resolution, shading, overlays)
- Save/load it through the existing preset system
- Use it when capturing previews
- Preset picker in the preview settings popover
