# ナビゲーションパネル (Navigation Panel)

**場所:** *Properties Editor (プロパティエディター) > Takes タブ*

ナビゲーションパネルは、あなたの全ての Takes 階層を管理するための第一のコントロールセンターです。ここには、Takes ツリー、カスケードのオーバーライドアサイン状況を示すアイコン、そしてバッチレンダリングへのアクセス機能が集約されています。

## ヘッダーコントロール (Header Controls)

パネルのヘッダー部分には現在のアドオンのバージョンが表示され、便利なクイックアクセスボタンが並んでいます：

| コントロール | 説明 |
|---------|-------------|
| **Lock (ロック)** | 全パネルの編集をロックし、誤操作による意図しない状態の変更を防ぎます。 |
| **Autokey (自動キーフレーム)** | すべてのシーンをまたいで、Blender 側の「自動キーフレーム (Auto-keying)」のオン/オフを一括で切り替えます。 |
| **Timeline Sync (タイムライン同期)** | 割り当てられた再生ヘッド (フレーム位置) を、すべてのシーン間で同期させます。 |
| **Warnings (警告表示)** | 発生中の危険な警告をハイライトします (リンク切れのプリセットや、失われかけている宙吊りのアクションなど)。 |
| **Globals (グローバル設定)** | カスケードの「グローバル」レベルにおける設定欄のみを開きます。 |
| **Help (ヘルプ)** | この公式ドキュメンテーション (Wiki) をブラウザで開きます。 |
| **Settings (表示設定)** | ツリーの可視性や UI のデザインオプションを開きます。 |

## Takes ツリー (The Takes Tree)

Takes ツリーは、あなたのプロジェクトの全構造を視覚的に結び付け、統合された一つの階層リストとして見せてくれます：

```
📁 Scene Group (シーン主グループ)
  🎬 Scene (シーン)
    📂 VL Group (VL小グループ)
      🔲 View Layer (ビューレイヤー)
        📌 VL Version (VL改訂版)
```

### 行のエレメント (Row Elements)

View Layer（ビューレイヤー）の各行には、カスケードのオーバーライド状態を示すアイコン類が表示されます。これらのアイコンを見れば、その階層レベルにおいて「どのプロパティが上書き設定（オーバーライド）されているか」が一目で分かります：

- **Ghost (ゴースト)** — レストステート (Rest State) の記録状況
- **Tag (タグ)** — 割り当てられたカラー識別タグ
- **Variant (バリアント)** — アクティブなバリアントスイッチ (Variant Switch) のステート指定
- **Action (アクション)** — カスケードによるアニメーションアクションの割り当て状況
- **Compositor (コンポジター)** — コンポジターのノードツリーの上書き状況
- **World (ワールド)** — ワールド/環境設定の上書き状況
- **Camera (カメラ)** — カメラの割り当て状況
- **Output (出力ルール)** — 出力時のパス/名付けルールの指定状況
- **Render (レンダー)** — レンダリングと画質のプリセット指定状況

!!! tip "カスケードアイコンの超過 (Overflow)"
    サイドバーのパネル幅が狭い場合、カスケードアイコンは自動的に折りたたまれて溢れ、
    **⋯** (オーバーフロー) ボタンとして表示されます。これをクリックすると、ポップオーバーメニューが開き、すべてのアイコンを完全に確認できます。

### ツリーのガイドライン (Tree Lines)

自由に設定可能なインデント線 (ツリーの階層を繋ぐ線) が、階層構造を視覚的に示してくれます。割り当てられた識別用タグの色を、このツリーのガイドライン自体に継承させて素早く見分けることも可能です。

## 表示設定ポップオーバー (Settings Popover)

**歯車アイコン** をクリックすると、ツリーの表示に関する設定にアクセスできます：

| 設定項目 | 説明 |
|---------|-------------|
| **Show Tag Color (タグカラー色分け)** | 割り当てられたカラータグの色を、ツリーのインデント線 (階層線) にも反映させます。 |
| **Line Width (線幅)** | インデント線の太さを Thin (細い) / Medium (普通) / Wide (太い) から選びます。 |
| **Icon Visibility (アイコンの可視性)** | 特定のカスケードアイコンの表示/非表示を個別に切り替えます。 |
| **Preview Size (プレビューサイズ)** | インラインVLプレビュー (サムネイル) のサイズを変更します (24/32/40 px)。 |

## 危険の警告システム (Warnings)

ナビゲーションパネルは、プロジェクトに深刻な問題が検出された場合に大々的に警告を表示します：

- **Missing Preset (プリセット消失)** — カスケードのプリセット参照先として指定された JSON ファイルがディスク上から削除されています。
- **Dangling Action (宙吊りのアクション)** — オブジェクトの「自動割り当て (Auto-Assign)」保護がオフになっているため、現在のアクションがシステムに吸収されず完全に消失しようとしています。
