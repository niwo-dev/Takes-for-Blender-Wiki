# Takes (テイクシステム)

**Take** (テイク) とは、あなたのシーンにおける特定の構成状態 ──すなわち、特有のカメラアングル、独自の照明セットアップ、特定の素材バリアント、あるいは固有のアニメーション進捗を含んだ、名付けられた「View Layer (ビューレイヤー) の小グループ」のことを指します。

## コンセプト

映画や写真撮影において、「テイク」とはそのショットのひとつのバージョンを意味します。Takes for Blender はこの概念を Blender の View Layer システム内部まで拡張し、あなたに以下の特権を与えます：

- View Layer ごとの **独立したカメラ割り当て**
- View Layer ごとの **独立したワールド環境割り当て**
- View Layer ごとの **独立したコンポジター用ノードツリー割り当て**
- View Layer ごとの **独立したレンダープリセット割り当て**
- View Layer ごとの **独立したアニメーション用アクション割り当て**

## トップダウンでの階層構造 (Organization)

それぞれの Take (テイク) は、Takes ツリーの階層内で組織的に管理されます：

| レベル階層 | 目的・用途 | 使用例 |
|-------|---------|---------|
| **Scene Group (シーン主組)** | 最上位の巨大なグルーピング組織 | 「インテリア全体」, 「外装全体」 |
| **Scene (シーン)** | Blender に元からある Scene | 「キッチン」, 「バスルーム」 |
| **VL Group (VL小組)** | VL群の意味合い・論理的なグループ化 | 「ヒーローショット集」, 「ディテール寄りカット集」 |
| **View Layer (VL本体)** | 実際のレンダリング出力を担う中心ユニット | 「斜め前方アングル 3/4」, 「真俯瞰ショット」 |
| **VL Version (VL改訂版)** | 特定VLの設定値に対する名付けられたスナップショット | 「v1 暖色系」, 「v2 寒色系」 |

## Take の作成手順

### View Layer の追加

1. ツリーサイドバー内にある **+** ボタンをクリックします。
2. **Add View Layer (ビューレイヤーの追加)** を選択します。
3. 作成された新しい VL は、現在アクティブであった View Layer から設定を継承します。

### View Layer のグループ化

1. ツリー内で任意の View Layer を選択します。
2. ++ctrl+g++ を押して、それを含む新しい「VL Group」を作成します。
3. 他の View Layer を、そのグループ内にドラッグして移すことができます。

### VL Version (改訂版スナップショット) の運用

View Layer のカスケード設定状態に対する、名前付きスナップショット (保存版) を作成します：

1. 目当ての View Layer を選択します。
2. **+** → **Add Version (バージョンの追加)** をクリックします。
3. 各バージョンは、それぞれ固有のカメラ、ワールド、アクション、およびプリセットの上書き設定を自分自身の中に保存 (退避) します。

!!! tip "瞬時の切り替え (Quick Switching)"
    VL のバージョン間を切り替えることで、大元の View Layer 自体をいくつも無駄に複製・増殖させずとも、
    異なる構成を素早く・即座に比較できるようになります。
