## Ключевые возможности

| Функция | Описание |
|---------|-------------|
| **Дерево (Takes Tree)** | Единая иерархия: Scene Groups → Scenes → VL Groups → View Layers → Versions. |
| **6-уровневый Каскад** | Разрешение переопределений от глобального уровня до уровня версии для камер, окружения, действий и т.д. |
| **Variant Switch** | Система вариантов продукта с заменой материалов, пресетов и управлением пулами. |
| **Rest State** | Система авто-отражения базового действия (reference action), сохраняющая исходные ключевые кадры при анимации. |
| **Batch Render** | Модальный движок рендера (на переднем или заднем плане) с системой тегов Smart Output для имен файлов. |
| **Render Presets** | JSON-пресеты для настроек рендера, камеры, окружения и вывода файлов — для каждого View Layer. |
| **Встроенные превью** | Иконки предпросмотра порта прямо в дереве рядом с каждым View Layer. |

!!! tip "Версия Blender"
    Takes for Blender требует **Blender 5.1** или новее. Он использует систему Slotted Action, представленную в Blender 5.0.
