# Установка

## Требования

- **Blender 5.1** или новее
- Windows, macOS или Linux

## Скачивание

Скачайте последний ZIP-архив релиза со страницы [GitHub Releases](https://github.com/niwo-dev/Takes-for-Blender/releases).

!!! warning "Не распаковывайте"
    Blender устанавливает аддоны напрямую из `.zip` файла. **Не** распаковывайте его перед установкой.

## Установка в Blender

1. Откройте Blender и перейдите в **Edit > Preferences > Add-ons**.
2. Нажмите **Install from Disk...** (выпадающее меню сверху справа).
3. Найдите скачанный `.zip` файл и выберите его.
4. Активируйте аддон, поставив галочку рядом с **Takes for Blender**.

## Проверка установки

После активации на боковой панели **Properties Editor** появится новая вкладка **Takes** (нажмите ++n++, чтобы переключить боковую панель, если она скрыта).

Вы должны увидеть:

- Панель навигации (**Navigation Panel**) с селекторами Take / View Layer
- Дерево снимков (**Takes Tree**), показывающее ваши текущие сцены и слои (view layers)

!!! tip "Первый запуск"
    При первом запуске с настройками по умолчанию аддон автоматически включает
    **Auto-Assign** и **Auto-Rename** для действий каскада.
    Это означает, что данные анимации управляются для каждого View Layer «из коробки».

## Обновление

Чтобы обновиться до более новой версии:

1. Перейдите в **Edit > Preferences > Add-ons**.
2. Найдите **Takes for Blender** и нажмите на стрелку выпадающего меню.
3. Нажмите **Remove** (Удалить).
4. Перезапустите Blender.
5. Выполните шаги установки выше с новым `.zip` файлом.

!!! note "Сохранение настроек"
    Ваши настройки аддона сохраняются в файл JSON и переносятся при обновлениях,
    пока включена опция **Autosave Preferences** (включена по умолчанию).
