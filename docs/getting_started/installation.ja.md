# インストール (Installation)

## 必要動作環境

- **Blender 5.1** またはそれ以上
- Windows, macOS, あるいは Linux

## ダウンロード

[GitHub Releases](https://github.com/niwo-dev/Takes-for-Blender/releases) ページから、最新のリリース ZIP ファイルをダウンロードしてください。

!!! warning "解凍 (Unzip) しないでください"
    Blender はアドオンを `.zip` ファイルから直接インストールします。インストールする前に絶対に中のファイルを解凍（展開）しないでください。

## Blender へのインストール

1. Blender を起動し、**Edit (編集) > Preferences (プリファレンス) > Add-ons (アドオン)** を開きます。
2. 右上のドロップダウンをクリックし、**Install from Disk... (ディスクからインストール...)** を選択します。
3. ダウンロードした `.zip` ファイルを辿って選択します。
4. **Takes for Blender** の横にあるチェックボックスにチェックを入れて、アドオンを有効化します。

## インストールの確認

有効化後、**Properties Editor (プロパティエディター)** のサイドバーに新しい **Takes** タブが表示されます（サイドバーが隠れている場合は、キーボードの ++n++ で表示の切り替えができます）。

以下のような画面が表示されるはずです：

- Take / View Layer (ビューレイヤー) のセレクターがある **ナビゲーションパネル (Navigation Panel)**
- 現在のシーンとビューレイヤーを表示する **Takes ツリー (Takes Tree)**

!!! tip "初回起動について"
    デフォルト設定での初回起動時には、アドオンがカスケードアクションの **Auto-Assign (自動割り当て)** と **Auto-Rename (自動リネーム)** を自動的に有効にします。
    これはつまり、アニメーションデータが、何も設定しなくても直ぐにビューレイヤーごとに管理・分離されることを意味します。

## アップデート

より新しいバージョンへアップデートするには：

1. **Edit (編集) > Preferences (プリファレンス) > Add-ons (アドオン)** を開きます。
2. **Takes for Blender** を見つけて、ドロップダウン矢印をクリックして開きます。
3. **Remove (削除)** をクリックして一度アドオンを消去します。
4. Blender を再起動します。
5. 新しいバージョンの `.zip` ファイルを使用して、上記のインストール手順をもう一度実行します。

!!! note "設定の永続性について"
    **Autosave Preferences (プリファレンスの自動保存)** が有効になっている限り（デフォルトで有効です）、あなたのアドオンの環境設定は JSON ファイルに保存され、アップデート後も自動的に維持されます。
