# Интерфейс

Интерфейс Takes for Blender располагается на боковой панели **Properties Editor** (Редактора свойств). Он состоит из четырех основных панелей, доступных через панель вкладок в верхней части Navigation Panel (Панели навигации).

## Обзор панелей

| Панель | Назначение |
|-------|---------|
| [**Панель навигации (Navigation Panel)**](navigation_panel.md) | Основной центр управления — переключение Take/VL, древовидное представление, значки каскада и доступ к пакетному рендеру. |
| [**Панель инспектора (Inspector Panel)**](inspector_panel.md) | Список отслеживания для каждого объекта, показывающий управляемые/закрепленные объекты, их действия, слоты и подданные (sub-data). |
| [**Свойства контекста (Context Properties)**](context_properties.md) | Свойства переопределений каскада для каждого View Layer — камеры, окружение, композиторы, действия и пресеты. |
| [**Дерево вариантов (Variant Tree)**](variant_tree.md) | Управление вариантами продукта — продукты, состояния, детали и пулы материалов. |

## Навигационные вкладки

Заголовок Панели навигации содержит три кнопки-переключателя (toggles) для смены видов Context, Link и Inspector:

- **Context** (Контекст) — Дерево Takes Tree и настройки для каждого VL (вкладка по умолчанию)
- **Link** (Связь) — Управление и связывание коллекций
- **Inspector** (Инспектор) — Список наблюдения за объектами и управление действиями/слотами

!!! tip "Горячие клавиши"
    Многие операции с деревом поддерживают сочетания клавиш клавиатуры:

    - ++ctrl+n++ — Добавить новый элемент (с учетом контекста дерева)
    - ++shift+d++ — Дублировать (полная независимая копия)
    - ++alt+d++ — Дублировать (связанная объединенная копия)
    - ++ctrl+g++ — Создать группу
    - ++alt+g++ — Разгруппировать выбранное
    - ++del++ или ++x++ — Удалить с предварительным подтверждением
    - ++f2++ — Переименовать текущий элемент
