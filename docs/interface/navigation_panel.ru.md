# Панель навигации (Navigation Panel)

**Расположение:** *Properties Editor (Свойства) > Вкладка Takes*

Панель навигации является основным центром управления вашей иерархией Takes. Она содержит Дерево Takes Tree, значки переопределения каскада и обеспечивает доступ к пакетному рендерингу (batch rendering).

## Элементы управления заголовка

Заголовок панели отображает текущую версию аддона и предоставляет кнопки быстрого доступа:

| Элемент управления | Описание |
|---------|-------------|
| **Lock (Блокировка)** | Блокирует панель во избежание случайных изменений состояния. |
| **Autokey (Авто-ключ)** | Переключает режим авто-сохранения ключей (auto-keying) Blender одновременно для всех сцен. |
| **Timeline Sync (Синхронизация шкалы)** | Синхронизирует позицию плейхеда (текущий кадр) для всех сцен. |
| **Warnings (Предупреждения)** | Отображает активные проблемы (отсутствующие пресеты, потерянные действия). |
| **Globals (Глобальные настройки)** | Открывает настройки каскада глобального уровня проекта. |
| **Help (Помощь)** | Открывает документацию (эту вики-страничку). |
| **Settings (Настройки)** | Опции видимости и дизайна дерева в навигации. |

## Дерево снимков (The Takes Tree)

Дерево Takes Tree — это единый иерархический список, показывающий всю структуру вашего проекта:

```
📁 Scene Group (Группа сцен)
  🎬 Scene (Сцена)
    📂 VL Group (Группа VL)
      🔲 View Layer (Слой просмотра)
        📌 VL Version (Версия VL)
```

### Элементы строк

В каждой строке View Layer отображаются иконки каскадных переопределений (cascade overrides). По этим иконкам можно сразу понять, какие свойства переопределены на данном уровне:

- **Ghost (Призрак)** — Статус состояния покоя (Rest State)
- **Tag (Тег)** — Назначенный цветовой тег
- **Variant (Вариант)** — Активное состояние переключателя вариантов
- **Action (Действие/Анимация)** — Назначение каскадного действия
- **Compositor (Композитор)** — Переопределение дерева нодов композитора
- **World (Окружение)** — Переопределение мирового окружения
- **Camera (Камера)** — Назначение активной камеры
- **Output (Вывод)** — Тег/правило вывода для рендера
- **Render (Рендер)** — Тег параметров рендера (Render preset)

!!! tip "Скрытие значков (Cascade Icon Overflow)"
    Когда панель слишком узкая, все значки каскада автоматически схлопываются в дополнительную скрытую кнопку **⋯**. При нажатии на неё открывается всплывающее окно (popover), где все иконки каскада видны полностью.

### Линии дерева

Настраиваемые линии отступов визуально отображают иерархию элементов. Эти линии могут автоматически окрашиваться в цвета назначенных тегов для быстрой идентификации.

## Всплывающее окно настроек (Settings Popover)

Щелкните по значку **шестеренки**, чтобы открыть параметры отображения дерева:

| Настройка | Описание |
|---------|-------------|
| **Show Tag Color (Цвет тега)** | Окрашивать линии отступа дерева в соответствии с назначенными тегами. |
| **Line Width (Ширина линии)** | Тонкие / Средние / Широкие линии отступов. |
| **Icon Visibility (Видимость значков)** | Включить/отключить видимость отдельных значков каскада для всего дерева. |
| **Preview Size (Размер превью)** | Размер миниатюр для встроенного предпросмотра слоя (24/32/40 пикселей). |

## Предупреждения

Панель навигации показывает предупреждения при обнаружении проблем:

- **Missing Preset (Отсутствующий пресет)** — Каскадная ссылка указывает на уже удаленный файл JSON.
- **Dangling Action (Потерянное действие)** — Активное действие (анимация) может быть потеряно из-за того, что функция автоматического назначения (Auto-Assign) выключена.
