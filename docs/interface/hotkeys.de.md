# Tastaturkürzel

Alle Tastenkombinationen sind aktiv, wenn sich der Cursor über der **Seitenleiste des 3D-Ansichtsfensters** (N-Panel) befindet.
Tastenkombinationen, die in mehreren Listen dieselbe Taste verwenden, sind **fokusabhängig** – es reagiert nur die aktuell sichtbare/aktive Liste.

---

### Allgemein (alle Listen)

| Tastenkombination | Aktion | Anmerkungen |
|----------|--------|-------|
| <kbd>Strg</kbd> + <kbd>I</kbd> | **Auswahl umkehren** | Nur aktiv, wenn der Mehrfachauswahlmodus aktiviert ist. Kehrt die aktuelle Auswahl in der fokussierten Liste um. |

---

### Navigationsbereich (Baumstruktur)

| Tastenkombination | Aktion | Anmerkungen |
|----------|--------|-------|
| <kbd>Entf</kbd> / <kbd>X</kbd> | **Element löschen** | Entfernt die ausgewählte Szene, Ansichtsebene oder Gruppe. Zeigt einen Bestätigungsdialog an. Im Mehrfachauswahlmodus werden alle ausgewählten Elemente gesammelt gelöscht. |
| <kbd>Strg</kbd> + <kbd>N</kbd> | **Hinzufügen** | Öffnet das Menü „Hinzufügen“, um eine neue Szene oder einen neuen Ansichtsebene zu erstellen. |
| <kbd>F2</kbd> | **Umbenennen** | Öffnet einen Umbenennungsdialog für das ausgewählte Element. |
| <kbd>Strg</kbd> + <kbd>G</kbd> | **Gruppieren** | Gruppiert die ausgewählten Elemente in eine Szenengruppe oder eine VL-Gruppe. |
| <kbd>Alt</kbd> + <kbd>G</kbd> | **Gruppierung aufheben** | Entfernt das ausgewählte Element aus seiner Gruppe. |
| <kbd>Strg</kbd> + <kbd>T</kbd> | **Neu zuweisen** | Öffnet den Dialog „Neu zuweisen“ für die ausgewählte Szene oder den ausgewählten Ansichtsebene. |
| <kbd>Strg</kbd> + <kbd>Umschalt</kbd> + <kbd>Alt</kbd> + <kbd>I</kbd> | **Referenz festlegen** | Legt die Standardreferenz für das ausgewählte Element fest. |

---

### Variantenbaum

| Tastenkombination | Aktion | Anmerkungen |
|----------|--------|-------|
| <kbd>Entf</kbd> / <kbd>X</kbd> | **Element entfernen** | Entfernt das ausgewählte Produkt, den ausgewählten Zustand, das ausgewählte Teil oder das ausgewählte Pool-Material. Erzwingt mindestens 1 Zustand und 1 Teil pro Produkt. |
| <kbd>Strg</kbd> + <kbd>N</kbd> | **Intelligentes Hinzufügen** | Kontextbezogenes Hinzufügen: Erkennt automatisch, was basierend auf der aktuellen Auswahl erstellt werden soll. Wenn eine Status-Kopfzeile oder ein Status ausgewählt ist, wird ein Status hinzugefügt. Wenn ein Teil oder ein Pool-Material ausgewählt ist, wird ein Pool-Material hinzugefügt. Wenn ein Produkt ausgewählt ist, wird das vollständige Hinzufügen-Menü angezeigt. |
| <kbd>F2</kbd> | **Umbenennen** | Öffnet einen Umbenennungsdialog für das ausgewählte Produkt, den ausgewählten Zustand oder das ausgewählte Teil. |

---

### Mehrfachauswahlmodus

Der Mehrfachauswahlmodus wird durch Klicken auf die Umschaltfläche **☐** in der Statuszeile einer beliebigen Liste aktiviert.

| Tastenkombination | Aktion | Anmerkungen |
|----------|--------|-------|
| <kbd>Umschalt</kbd> + Klick (Umschalten) | **Alle auswählen** | Wählt alle sichtbaren Elemente aus. Berücksichtigt aktive Typfilter. |
| <kbd>Alt</kbd> + Klick (Umschalten) | **Umkehren** | Kehrt die aktuelle Auswahl um. |
| <kbd>Strg</kbd> + <kbd>I</kbd> | **Umkehren** | Wie Alt+Klick – Tastenkombination zum Umkehren. |
| Auf Statistik-Symbol klicken | **Typfilter** | Filtert die Kontrollkästchen nach einem bestimmten Typ (z. B. nur Szenen, nur Zustände). Erneut klicken, um zu löschen. |
