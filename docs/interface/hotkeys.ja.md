# キーボードショートカット (Keyboard Shortcuts)

すべてのホットキーは、マウスカーソルが **3D Viewport サイドバー** (Nボタンによるサイドパネル) の上にあるときにのみアクティブになります。  
複数のリスト間で同一のキーに割り当てられているショートカットは、全て **フォーカス認識型 (focus-aware)** の設計となっており、現在あなたが視覚的に開いている (あるいはアクティブ化している) リストに対してのみ反応し、無関係な誤動作を起こすことはありません。

---

### ジェネリック用途：全共通 (Generic - All Lists)

| ショートカット | 発生アクション | 詳細・注記 |
|----------|--------|-------|
| <kbd>Ctrl</kbd> + <kbd>I</kbd> | **Invert Selection (選択を反転)** | 複数選択モード (multi-select mode) が有効になっている場合にのみ作動します。フォーカスされている現在のリスト全体で、チェック対象の選択状況を「全反転」させます。 |

---

### ナビゲーションパネル：Takes ツリー (Navigation Panel - Takes Tree)

| ショートカット | 発生アクション | 詳細・注記 |
|----------|--------|-------|
| <kbd>DEL</kbd> / <kbd>X</kbd> | **Delete Item (アイテムの削除)** | 選択されているその当該 Scene、View Layer、または各種 Groups 全体を削除します (確認ダイアログの警告が出ます)。複数選択モードが有効な場合は、チェックされたすべてを一括でバッチ的に削除します。 |
| <kbd>Ctrl</kbd> + <kbd>N</kbd> | **Add (新規追加)** | 最も文脈に沿った新規の Scene または View Layer を作成するための「Add メニュー」を開きます。 |
| <kbd>F2</kbd> | **Rename (リネーム / 名前の変更)** | 選択されているアイテム専用の、名前変更ダイアログボックスを開きます。 |
| <kbd>Ctrl</kbd> + <kbd>G</kbd> | **Group (グループへの梱包)** | 選択したアイテムらを、特設した Scene Group または VL Group の中に直ぐにグループ化してまとめることができます。 |
| <kbd>Alt</kbd> + <kbd>G</kbd> | **Ungroup (グループからの解除)** | それらのグループに所属している特定のアイテムを、グループの拘束から引き抜いて解除します。 |
| <kbd>Ctrl</kbd> + <kbd>T</kbd> | **Retarget (リターゲット再照準)** | 選択されている当該 Scene や View Layer のための、ターゲット再指定ダイアログを強制的に開きます。 |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Alt</kbd> + <kbd>I</kbd> | **Set Reference (独自の基準位置へ設定)** | 選択した対象のアイテムを、「自身のデフォルト基準値（リファレンスデフォルト値）」として設定します。 |

### バリアントツリー (Variant Tree)

| ショートカット | 発生アクション | 詳細・注記 |
|----------|--------|-------|
| <kbd>DEL</kbd> / <kbd>X</kbd> | **Remove Item (アイテム取り除き・削除)** | 選択された Product や State 本体、各 Part 部品、または割り当てた プール用マテリアル などを削除します。ただしシステム保護のため、Product 1つにつき、最低「1つのState」と「1つのPart」を常に維持しようとする制限が掛かります。 |
| <kbd>Ctrl</kbd> + <kbd>N</kbd> | **Smart Add (文脈追随型のスマート追加作成)** | 完全なコンテキスト認識型の追加システムです。: 現在選択している要素を元に、何を新しく作るべきか"気を利かせて"自動検出します。もしState層に居れば新しい「State」を追加します。Part や プールマテリアルを選択していれば、代わりに「プールマテリアルの枠」を追加させます。そして親玉である Product を選択していれば、フルの追加メニューを表示させる等、極めてスマートに機能します。 |
| <kbd>F2</kbd> | **Rename (リネーム / 名前の変更)** | 選択した Product、State、ないしは Part 部品名称のリネーム用ダイアログを開きます。 |

---

### 複数選択モード専用の動作 (Multi-Select Mode)

複数選択モードを使用するためには、いずれかのリスト枠の横一番下（統計行 / stats row）に位置する **☐** トグルアイコンをクリックしてアクティブ化しておく必要があります。

| ショートカット | 発生アクション | 詳細・注記 |
|----------|--------|-------|
| <kbd>Shift</kbd> + クリック操作 (切り替え) | **Select All (すべて選択)** | 目に見えるリスト上のすべてのアイテムに一括チェックを入れます。その際に、あなたが有効にしている「タイプ・フィルター」の状態も完全に尊重します。 |
| <kbd>Alt</kbd> + クリック操作 (切り替え) | **Invert (反転)** | 現在入っているチェック状態を一括で瞬時に反転させます。 |
| <kbd>Ctrl</kbd> + <kbd>I</kbd> | **Invert (反転 / キー仕様)** | 上記 Alt+Click（クリック式）と同じアクションを発生させるための、純粋なキーボード側のショートカット版です。 |
| 対象のアイコンをクリック | **Type Filter (対象で絞り込むフィルター)** | 全表示リストを特定の「タイプ」のみに絞り込ませます。(例：純粋なScenes行だけ残す、あるいはStates行だけを抽出する等)。もう一度そのアイコンをクリックすると、フィルターはクリアされ解除されます。 |
