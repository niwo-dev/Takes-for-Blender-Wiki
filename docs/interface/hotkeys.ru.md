# Горячие клавиши (Keyboard Shortcuts)

Все горячие клавиши активны и работоспособны, когда курсор мыши находится над **боковой панелью 3D-вьюпорта** (N-panel).
Горячие клавиши, которые в разных списках используют одну и ту же кнопку, знают о **компоненте с фокусом ввода (focus-aware)** — на них будет реагировать только видимый в данный момент или активный список элементов.

---

### Общие клавиши (Для любых списков)

| Сочетание | Команда | Описание |
|----------|--------|-------|
| <kbd>Ctrl</kbd> + <kbd>I</kbd> | **Инвертировать выделение** | Активно только в режиме множественного выбора (multi-select). Инвертирует текущее выделение во всем активном списке, находящемся в фокусе. |

---

### Панель навигации (Для дерева Takes Tree)

| Сочетание | Команда | Описание |
|----------|--------|-------|
| <kbd>DEL</kbd> / <kbd>X</kbd> | **Удалить элемент** | Удаляет выбранную сцену (Scene), View Layer или Группу (Group). Всегда показывает дополнительный диалог подтверждения. В режиме мультивыбора производит пакетное удаление всех выделенных объектов. |
| <kbd>Ctrl</kbd> + <kbd>N</kbd> | **Добавить** | Открывает меню добавления (Add menu) для создания новой сцены Сцены или View Layer. |
| <kbd>F2</kbd> | **Переименовать** | Открывает окно для переименования имени у текущего элемента. |
| <kbd>Ctrl</kbd> + <kbd>G</kbd> | **Группировать** | Создает Группу (Scene Group / VL Group) из текущих элементов списка. |
| <kbd>Alt</kbd> + <kbd>G</kbd> | **Разгруппировать** | Извлекает выбранный в данный момент элемент из группы. |
| <kbd>Ctrl</kbd> + <kbd>T</kbd> | **Перенацеливание** | Вызывает мощный диалог ретаргетинга (retarget dialog) для выбранной сцены или View Layer. |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Alt</kbd> + <kbd>I</kbd> | **Задать референс базой** | Применяет значение базового референса по умолчанию (reference default) для выбранного текущего объекта. |

---

### Дерево вариантов (Для Variant Tree)

| Сочетание | Команда | Описание |
|----------|--------|-------|
| <kbd>DEL</kbd> / <kbd>X</kbd> | **Удалить деталь/вариант** | Удаляет выбранные Продукты (Product), Состояния (State), Детали (Part) или Материалы пула. Всегда строго заставляет оставлять не менее 1 Состояния (State) и 1 Детали (Part) внутри каждого Продукта. |
| <kbd>Ctrl</kbd> + <kbd>N</kbd> | **Умное добавление** | Контекстное добавление: автоматически определяет, что следует добавить, основываясь на вашем выборе. При выборе заголовка состояний (State header) или самого состояния (State) - будет добавлять состояние (State). При выборе Детали (Part) или материала из пула (Pool Material) - добавит новый материал (Pool Material). Для Продуктов (Product) развернет полное меню доступных пунктов. |
| <kbd>F2</kbd> | **Переименовать** | Открывает окно-слайдер для переименования активных позиций Product, State или Part. |

---

### Режим мульти-выделения (Multi-Select Mode)

Режим с множественным выбором списка или объектов активируется путем физического нажатия или включения кнопки с иконкой-флажком **☐** (toggle), находящейся в самом низу статистической и информационной строки в любом доступном списке.

| Сочетание | Команда | Описание |
|----------|--------|-------|
| <kbd>Shift</kbd> + Левый клик мыши | **Выбрать всё** | Выбирает и отмечает все (видимые) объекты в списке. При этом игнорирует позиции для скрытых фильтров. |
| <kbd>Alt</kbd> + Левый клик мыши | **Инверсия выбора** | Осуществляет мгновенную инверсию вашего последнего операционного выбора флажков. |
| <kbd>Ctrl</kbd> + <kbd>I</kbd> | **Инверсия выбора** | Тот же функционал, что и комбинация Alt+Click — но только с использованием стандартного горячего шортката (горячих клавиш) системы. |
| Нажать на иконку типа ресурса (stats) | **Фильтр типа** | Активно фильтрует флажки для определенного типа (например, покажет для выделения только сцены, или покажет только состояния - 'States'). Нажмите повторно на иконку фильтра, чтобы обнулить фильтрацию. |
