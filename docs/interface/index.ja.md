# インターフェース (Interface)

Takes for Blender のインターフェースは、基本 **Properties Editor (プロパティエディター)** のサイドバー（Nパネル）内に配置されています。Navigation Panel (ナビゲーションパネル) 上部にあるタブバーからアクセスできる 4つの主要なパネルで構成されています。

## パネルの概要

| パネル | 目的 |
|-------|---------|
| [**ナビゲーションパネル (Navigation Panel)**](navigation_panel.md) | 主要なコントロールセンター。Take/VLの切り替え、ツリービュー、各種カスケードアイコンの視認、そしてバッチレンダリングへのアクセスを担います。 |
| [**インスペクターパネル (Inspector Panel)**](inspector_panel.md) | オブジェクトごとの監視リスト (ウォッチリスト)。自動管理下にあるオブジェクトやピン留めされたオブジェクト、そしてそれらに割り当てられたアクション、スロット、サブデータ（マテリアル等）を細かく表示・統括管理します。 |
| [**コンテキストプロパティ (Context Properties)**](context_properties.md) | ビューレイヤー (View Layer) ごとのカスケード権限オーバーライド確認パネル。カメラ、ワールド、コンポジター、アクション、プリセット等の詳細を扱います。 |
| [**バリアントツリー (Variant Tree)**](variant_tree.md) | 製品のバリアント (異なる仕様や素材の組み合わせ等) を管理するパネル。プロダクト本体、ステート（状態派生）、パーツ構成、及び利用可能なマテリアルプールを管理します。 |

## ナビゲーションタブ (Navigation Tabs)

ナビゲーションパネルのヘッダー部分には、Context (コンテキスト)、Link (リンク)、Inspector (インスペクター) の3つのビューを切り替えるためのトグルボタンが含まれています：

- **Context** — Takes ツリー本体と、VL ごとの設定を表示
- **Link** — コレクション (Collection) のリンク状態と管理画面
- **Inspector** — オブジェクトのウォッチリストと、アクション/スロットの管理画面

!!! tip "キーボードショートカットについて"
    ツリー上の多くの操作は、以下のキーボードショートカットに対応しています：

    - ++ctrl+n++ — 新規アイテムの追加 (※何を選択しているかによって文脈を自動感知します)
    - ++shift+d++ — 複製 (完全な独立コピー)
    - ++alt+d++ — 複製 (リンクされたコピー)
    - ++ctrl+g++ — グループ化 (Group)
    - ++alt+g++ — グループ解除 (Ungroup)
    - ++del++ または ++x++ — 確認ダイアログ付きの削除
    - ++f2++ — 名前の変更 (リネーム)
