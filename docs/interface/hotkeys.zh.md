# 键盘快捷键 (Keyboard Shortcuts)

为了实现无缝作业，所有的热键均需将你的鼠标停泊滞留在 **3D 视图内的右侧面板边缘地带** 上方时响应并唤醒。
因为相同的快捷按板会在多块系统逻辑面板内公用和交融，我们采取了极严密的聚焦响应 (**focus-aware**) 处理，只有此时时刻当前可见面板且拥有选定焦点的区块内容作出动作反应。

---

### 通用键位组合 (Generic All Lists)

| 按键方式 | 指令 | 注释提示 |
|----------|--------|-------|
| <kbd>Ctrl</kbd> + <kbd>I</kbd> | **反向选取 (Invert)** | 该组合专用于复选框多开模式激活之时 (Multi-Select)，直接颠倒勾选列表中相反之未中选项功能状态。 |

---

### Takes 导航与主架构树 (Navigation Panel)

| 按键方式 | 指令 | 注释提示 |
|----------|--------|-------|
| <kbd>DEL</kbd> 或 <kbd>X</kbd> | **移除废除 (Delete)** | 直接摧毁掉已被点中选取的目标 Scene，一层 View Layer 抑或是整体 Groups，此招因为过分危险肯定会唤起一道确定警示拦下保护。如果在多点选模式下便会自动成批摧毁删除所钩之复数物。 |
| <kbd>Ctrl</kbd> + <kbd>N</kbd> | **新鲜唤生 (Add)** | 直接弹出快捷窗生成最新 Scene(场景级) 或者 VL 层。 |
| <kbd>F2</kbd> | **更迭其名 (Rename)** | 叫出一条行内短小输入表单进行任意选中名字字符改写。 |
| <kbd>Ctrl</kbd> + <kbd>G</kbd> | **打组套牢 (Group)** | 在当前同属类型里，集结打包组建全新的文件夹套。 |
| <kbd>Alt</kbd> + <kbd>G</kbd> | **解绑释放 (Ungroup)** | 把被圈入其中单独选中的家伙，从中强制脱出拉离释放出文件夹架构中恢复顶层独立户。 |
| <kbd>Ctrl</kbd> + <kbd>T</kbd> | **标靶复指 (Retarget)** | 当某些特定层不巧丢失原有源指向或者要移指到另个目标时快速定位开启靶窗 (retarget)。 |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Alt</kbd> + <kbd>I</kbd> | **设为初基 (Set Ref)** | 固定该目标物当前属性信息当作为静止初始 (Reference State) 状态点。 |

---

### 变体体系分支 (Variant Tree)

| 按键方式 | 指令 | 注释提示 |
|----------|--------|-------|
| <kbd>DEL</kbd> 或 <kbd>X</kbd> | **抹灭变体项** | 从架构里去除一切挑出来的状态与产品，然而此举强行底线上至少各层会帮你兜底卡死至少不低于1个存活底线来免系统崩溃瘫痪的危险。 |
| <kbd>Ctrl</kbd> + <kbd>N</kbd> | **智能机变录入** | 高深之操作: 一眼探测目前正停泊鼠标和点选了哪项大类别系统便全方位跟随判断该当诞生啥儿子给你，若在产品便开所有选单，在款下给你塞子节点，完全全自动化适配对应同层级别建档不拖泥带水。 |
| <kbd>F2</kbd> | **重置起名** | 老规矩，轻巧修改各个款或者各条主脉的字形代号标识。 |

---

### 批量多选控制面板

必须去单独摸下并按压点亮树状底部的那个框选按钮 **☐** 将系统置位启动之后才有以下后续神妙化学反应使用权力。

| 按键方式 | 指令 | 注释提示 |
|----------|--------|-------|
| 依靠 <kbd>Shift</kbd> + 戳左键 | **一网收尽 (All)** | 把当下视野内能看见的人员不顾身份直接圈点齐集上阵，无视中间阻隔。（遵循受当区过滤设定等）。 |
| 搭配 <kbd>Alt</kbd> + 左键拍打 | **颠倒黑白 (Inv)** | 和上面的 <kbd>Ctrl</kbd> + <kbd>I</kbd> 操作出奇完全形合一的绝技颠覆现有盘面逻辑框点对象。 |
| 直接击拿最底边各类形图符 | **精确打击之筛 (Filter)** | 直接将整个复选择区内的打勾锁定至某种特有同类属大科目上去 (比如"给我只把当前属于Scene场景大项勾选的保留，其余勾去掉")。若再按一次则清空该层魔法回复。 |
