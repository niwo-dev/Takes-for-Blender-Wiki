# Características (Features)

Takes for Blender proporciona un conjunto de herramientas profundo para gestionar pipelines complejos de visualización de productos. Cada función está diseñada para trabajar conjuntamente a través del sistema cascade.

| Característica | Descripción |
|---------|-------------|
| [**Takes**](takes.md) | Grupos de View Layer con organización jerárquica. |
| [**Cascade System**](cascade.md) | Motor de resolución de anulaciones de propiedades de 6 niveles (tiers). |
| [**Variant Switch**](variant_switch.md) | Intercambio de materiales por contexto y variantes de producto. |
| [**Batch Render**](batch_render.md) | Renderizado automatizado multi-VL con gestión de colas. |
| [**Render Presets**](render_presets.md) | Preajustes basados en JSON para renderización, cámara, mundo e hitos de salida. |
| [**Rest State**](rest_state.md) | Acción de referencia con espejo automático para preservar keyframes predeterminados. |
| [**Smart Output**](smart_output.md) | Resolución dinámica de rutas de archivo basada en tokens. |
