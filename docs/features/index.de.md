# Funktionen

Takes for Blender bietet ein umfangreiches Toolset zur Verwaltung komplexer Pipelines für die Produktvisualisierung. Alle Funktionen sind so konzipiert, dass sie über das Kaskadensystem zusammenwirken.

| Funktion | Beschreibung |
|---------|-------------|
| [**Takes**](takes.md) | Ansicht von Layer-Gruppen mit hierarchischer Struktur. |
| [**Kaskadensystem**](cascade.md) | 6-stufige Engine zur Auflösung von Eigenschaftsüberschreibungen. |
| [**Variantenumschaltung**](variant_switch.md) | Materialwechsel und Produktvarianten pro Bereich. |
| [**Batch-Rendering**](batch_render.md) | Automatisiertes Multi-VL-Rendering mit Warteschlangenverwaltung. |
| [**Render-Voreinstellungen**](render_presets.md) | JSON-basierte Voreinstellungen für Render-, Kamera-, Welt- und Ausgabeeinstellungen. |
| [**Rest State**](rest_state.md) | Automatische Spiegelungsreferenzaktion zum Beibehalten von Standard-Keyframes. |
| [**Smart Output**](smart_output.md) | Dynamische, tokenbasierte Auflösung von Dateipfaden. |
