# Умный вывод (Smart Output)

Система **Smart Output** (Умный вывод) предоставляет динамическое разрешение путей к файлам для результатов рендера, на основе токенов (тегов). Вместо того чтобы вручную переименовывать и задавать путь каждому рендеру, токены (tags) в шаблоне автоматически заменяются на правильные, соответствующие текущему контексту значения.

## Включение системы Smart Output

1. Перейдите во вкладку **Output (Вывод)** в редакторе свойств Blender (Properties Editor).
2. Поставьте галочку активируя переключатель **Smart Output**.
3. Установите ваш предпочтительный **Шаблон директории (Directory Pattern)** и **Шаблон имени файла (File Name Pattern)** прямо в поле ввода, применяя систему встроенных токенов (меток).

## Синтаксис Токенов (Метки)

Каждый токен обрамляется специальными настраиваемыми скобками. По умолчанию, аддон поставляет стиль кодировки с использованием квадратных строгих скобок (brackets):

```
[view_layer]_[camera]_####.[file_format]
```

Во время старта задачи это строковое имя безошибочно генерируется компьютером во что-либо подобное: `Front_3-4_CamHero_0001.png`

### Стили Скобок (Bracket Styles)

Вы можете переключить ваш собственный, более привычный или любимый стиль форматирования внутри глобальных настроек **Addon Preferences > Behavior Options > Syntax Brackets**:

| Стиль | Синтаксис | Пример визуализации |
|-------|--------|---------|
| Квадратные | `[token]` | `[view_layer]_####` |
| Фигурные | `{token}` | `{view_layer}_####` |
| Угловые | `<token>` | `<view_layer>_####` |
| Круглые | `(token)` | `(view_layer)_####` |
| Знак процента| `%token%` | `%view_layer%_####` |
| Знак доллара | `$token$` | `$view_layer$_####` |
| Хэштеги | `#token#` | `#view_layer#_####` |

## Доступные Токены 

### Токены контекста (Context Tokens)

| Токен (Ввод) | Что он разрешает в тексте | Пример выхода |
|-------|-------------|---------|
| `scene` | Имя активной сцены | `Kitchen` |
| `view_layer` | Имя активного View Layer | `Front_3-4` |
| `camera` | Имя активной камеры слоя | `CamHero` |
| `group` | Название (имя) VL Группы | `Hero_Shots` |

### Токены анимации (Animation Tokens)

| Токен (Ввод) | Что он разрешает в тексте | Пример выхода |
|-------|-------------|---------|
| `frame` | Номер текущего кадра на шкале | `42` |
| `frame_start` | Стартовый (начальный) кадр сцены | `1` |
| `frame_end` | Конечный кадр сцены | `250` |
| `frame_range` | Диапазон "старт-конец" | `1-250` |
| `action` | Активное действие (анимация) сцены | `CameraFly` |
| `tks_action` | Активное назначенное каскадом действие | `SpinX` |

### Токены параметров Рендера (Render Tokens)

| Токен (Ввод) | Что он разрешает в тексте | Пример выхода |
|-------|-------------|---------|
| `engine` | Имя движка для рендера | `cycles` |
| `samples` | Количество сэмплов | `128` |
| `res_x` | Ширина разрешения (Ось Х) | `1920` |
| `res_y` | Высота разрешения (Ось У) | `1080` |
| `file_format` | Главный формат файла | `png` |

### Системные токены (System Tokens)

| Токен (Ввод) | Что он разрешает в тексте | Пример выхода |
|-------|-------------|---------|
| `blend` | Исходное имя Blend проекта | `project` |
| `date` | Полная текущая дата | `2026-03-29` |
| `time` | Точное локальное время | `14-30-00` |
| `sep` | Использование знака-сепаратора | `_` |

!!! tip "Таблица-Шпаргалка (Cheat Sheet)"
    Для вызова справочника нажмите кнопку **Syntax Tokens (Значок книги)** на верхней панели раздела свойств вывода. Он вызовет интерактивный список токенов и правил с их категориями.

## Разделители (Separators)

Вы можете использовать до 5 независимых видов различных знаков-разделителей (`[sep1]` до `[sep5]`) для разграничения информации (различные delimiter tokens):

- `[sep1]` = `_` обычно используется для имени файлов или папок
- `[sep2]` = `-` для логических переменных
- `[sep3]` = `.` для версии документа или файлов

Конфигурация (переназначение знаков препинания) для всех видов сепараторов проводится в глобальных настройках: **Addon Preferences**.

## Номера кадров (Frame Numbers)

Аддон Takes for Blender полномасштабно поддерживает нативный встроенный синтаксис Blender для кадров (символы `####` для длины и нулей). Smart Output автоматически обрабатывает такие символы как родные, корректно подставляя нулевые подстановки напрямую. Вы увидите правильную их подстановку прямо в строке предпросмотра (live preview).
