# Функции (Features)

Takes for Blender предоставляет глубокий набор инструментов для управления сложными конвейерами (pipelines) визуализации продуктов. Каждая функция разработана для совместной работы в единой каскадной системе.

| Функция | Описание |
|---------|-------------|
| [**Снимки (Takes)**](takes.md) | Группы View Layer с иерархической организацией. |
| [**Система Каскада (Cascade System)**](cascade.md) | 6-уровневый движок разрешения переопределений свойств. |
| [**Переключатель вариантов (Variant Switch)**](variant_switch.md) | Замена материалов на выбранном уровне и управление вариантами продуктов. |
| [**Пакетный рендер (Batch Render)**](batch_render.md) | Автоматизированный рендеринг нескольких VL с управлением очередью. |
| [**Пресеты рендера (Render Presets)**](render_presets.md) | JSON-пресеты для настроек вывода рендера, камеры, окружения (world) и других файлов. |
| [**Состояние покоя (Rest State)**](rest_state.md) | Авто-зеркалирование базового действия (reference action) для сохранения исходных ключей анимации по умолчанию. |
| [**Умный вывод (Smart Output)**](smart_output.md) | Динамическое разрешение путей к файлам на основе тегов (токенов). |
