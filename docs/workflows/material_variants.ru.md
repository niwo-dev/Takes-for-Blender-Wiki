# Рабочий процесс: Варианты материалов (Material Variants)

Данный рабочий процесс детально объяснит, как настроить разнообразные варианты цветов/отделки продукта при помощи использования многофункциональной системы "Variant Switch" (Переключатель вариантов).

## Сценарио 

Вы имеете продукт (например, бутылка для воды), который очень важно отрендерить в трёх совершенно уникальных вариантах отделки: "Матовый Черный", "Алюминий с матовой шлифовкой" и "Розовое Золото".

## Подготовка сцены

### 1. Создание Продукта

1. Откройте панельку **Variant Switch** на боковой панели.
2. Нажмите кнопку с плюсом **+**, чтобы создать новый пустой Продукт, назовите его "Бутылка (Bottle)".
3. Деталь (Part) по умолчанию, именуемая как "base" (основа) сгенерируется сама собой автоматически.

### 2. Задание Деталей (Parts)

Разделите ваш продукт на логические компоненты:

1. Добавьте Деталь и назовите ее "Body (Корпус)" → Выберите (привяжите) к ней коллекцию в которой физически хранится модель корпуса.
2. Добавьте Деталь под названием "Cap (Крышка)" → Привяжите соответствующую коллекцию крышки.
3. Добавьте Деталь "Label (Этикетка)" → И вновь привяжите её физическую коллекцию.

### 3. Добавление материалов в Пулы

Для абсолютно каждой физической детали пополните её локальный список материалов (pool):

1. Разверните меню детали "Body", чтобы добраться до её пула.
2. Назначьте свой скачанный или созданный материал `Mat_Matte_Black` прямо в свободный пустой слот 1.
3. Назначьте материал `Mat_Aluminum` во вновь появившийся пустой слот 2.
4. Назначьте материал `Mat_Rose_Gold` в автоматически появившийся слот 3.
5. Повторите эти шаги по аналогии для Крышки и Этикетки с их собственными, персональными материалами.

### 4. Создание Состояний (States)

Теперь проиндексируйте варианты:

1. Добавьте вверху Состояние "Matte Black" → И теперь укажите для каждой детали, чтобы оно брало свойства из **слота номер 1 (пул индекс 1)**.
2. Создайте Состояние "Aluminum" → Укажите использование **индекса 2** для всех деталей.
3. Состояние "Rose Gold" → Индекс пула номер **3**.

## Предварительный просмотр (Previewing)

Нажмите на **иконку ромбика (diamond icon)** над любым Состоянием в панели, чтобы незамедлительно применить все эти выбранные материалы разом на вашей 3Д-сцене во Viewport.

## Рендеринг всех вариантов

### Способ А: Обычный ручной

1. Активируйте Состояние (State) вручную.
2. Отрендерьте нажатием стандартной клавиши ++f12++.

### Способ Б: Интеграция с Каскадом (Cascade)

1. Просто создайте отдельные слои (View Layer) для каждого из необходимых вариантов.
2. Для каждого каскадного свойства VL в его строке назначьте (привяжите) этот конкретный вариант из "Variant Switch".
3. Вызовите Пакетный рендер (Batch Render), который самостоятельно просчитает все VL одним заходом без вашего вмешательства.

### Способ C: Комбинирование с ракурсами камеры

Создайте отдельный VL для каждой необходимой комбинации (уникальный вариант × уникальный ракурс камеры):

| Название View Layer | Вариант Материала (Variant) | Назначение Камеры |
|------------|---------|--------|
| Black_Front | Matte Black | Cam_Front |
| Black_Side | Matte Black | Cam_Side |
| Alu_Front | Aluminum | Cam_Front |
| Alu_Side | Aluminum | Cam_Side |

И наконец, пакетный рендер легко переварит весь массив этих комбинаций, а токены системы Умного Вывода (Smart Output) назовут это всё идеальными именами файлов:

```
[variant]_[camera]_####.[file_format]
```
