# 工作流：材质变体与换装 (Material Variants)

这个篇目咱们专门开搞讲解怎么操作那神奇伟大的 "变色龙换装换皮" 大法 (Variant Switch)，以此应付不同产品的多材质多规格颜色选项派单！

## 实战布景需求 (Scenario)

譬如说，你这手现在扣了一项目产品主秀（比如一只高逼格的保温水瓶），而金主方老爹极其要命要你分别对口三套全然做过手脚的不同表面加工工艺去呈现图纸: 其分明唤 "磨砂炭黑", "拉丝原铝" 以及 "尊贵粉玫瑰金" ！

## 预建立基础基建搭框架

### 1. 创立商品立项 (Create the Product)

1. 点击开启专用 **Variant Switch** 面板。
2. 立刻猛戳 **+** (或者叫新立点) 给建立只全新大项目，取个大称呼给它命字："大保温瓶 (Bottle)"。
3. 那此时此刻一款基本底座附送部件组件号叫做 ("base") 便被一早主动生成预装于该列队表末! 

### 2. 精密细分总构件法 (Define Parts)

咱们把它按组成部位分解卸装拆掉它:

1. 建个细部件代号唤： "Body / 筒身" → 顺手把你模型集合里的筒身这大集挂过来认亲。
2. 同法新建名： "Cap / 头盖" → 老大照样也拿模型头盖集合来结拜。
3. 把最后的标签牌立个名叫："Label / 小标签" → 收下最后它应归顺的对象组列。

### 3. 上满弹料库存 (Add Materials to Pools)

在这步骤，为你每一枚上面设编的特供专职组件全个去打进它未来所需预先挂齐料的充盈库存：

1. 往下掀掉这个属于 "Body筒子"的底裤看看它里面的存底材料箱 (pool)。
2. 空身就位第一格！马上放进去咱们的第一款颜色皮肤: 磨砂黑 `Mat_Matte_Black` ！进这插槽的1号。
3. 再下一槽呢，系统这会自动往下生出一个，赶快放 `Mat_Aluminum` 塞它的2位空洞! 
4. 如法炮制，再将最终那道 `Mat_Rose_Gold` 大菜落入 3槽。
5. 后面那些啥子瓶盖啦包边标单啦，各回各自列柜循规倒挂各自独有套料装备进同号位即好！ 

### 4. 列定出单品换装总号大名 (Create States)

要立案那些大名鼎鼎主打派变体款式咯:

1. 先建只全服叫 "尊享受炭黑(Matte Black)" 之大的 State 牌 → 那你叫它去取拿属于 1 号这个槽（pool index 1）里的货品全数统合强换上身！
2. 次列新加 "铝之本原(Aluminum)" State 头牌 → 命令把大权给到去拿走全阵营标排作 2 级号池内（pool index 2）的东西！
3. ...剩下的全叫它一率通往 3 这个字牌索引领料！ 

## 上眼！预览开看效期成绩 (Previewing)

啥废话也甭再搭腔讲了，立刻猛摁任何停驻前面那大大的那两块不被充色的钻石型徽章按钮去触发开启!（就上面挂在各个 State 外面那个）立竿见影！直接看效果上你的视窗大视口马上给你变！

## 各等全家福收列大全出图 (Rendering All Variants)

### 方法手段 A: 下等苦力古法流 (Manual)

1. 按顺序全手动点击点通变亮某个款子 State 。
2. 拍死 ++f12++ 看其自己产图。如此往复而已...

### 方法手段 B: 联动借取超级下落流水排法 (Cascade Integration)

1. 有几个总款型号数我给你建同样数量级的 View Layer 数来站门。
2. 点进去每户 View Layer 各门各户人家特带它那个换色变装牌子去(去跟那个叫 Variant Switch state 指定套上)。
3. 上总批量引擎发包 Batch Render 连串把它门收个总管！

### 方法手段 C: 十项全能结合相机流套用大圆满 

如果你那老爹（客户甲方）真逼你更进一步发大疯：既要多维度变装换颜又偏还要配带各种相向角度齐齐展示！组合乘积阵法开摆法：

| 上架大列表 VL | 拿哪家挂服色 (Variant) | 牵谁出镜头拍录 (Camera) |
|------------|---------|--------|
| 黑子大哥在前脸机(Black_Front) | 开挂碳黑装(Matte Black) | 首位打正前正面炮管(Cam_Front) |
| 黑子侧围边打光(Black_Side) | 还是那套原班碳黑着(Matte Black) | 这转开拉旁门侧影拍了(Cam_Side) |
| 换银将挂首前拍(Alu_Front) | 大铝本面装(Aluminum) | 打起排头冲面来着(Cam_Front) |
| 小光头侧开跑影(Alu_Side) | 连回它自身大铝面版(Aluminum) | 旁开另机给老二侧照相(Cam_Side) |

咱们底下包的无形批量算器将带你那大作一通天衣绝密结合组合统管生成，拿老一早设给智能代换导出机 (Smart Output tokens)这帮管家接包分名即可得大安： 
```
[variant]_[camera]_####.[file_format]
```
