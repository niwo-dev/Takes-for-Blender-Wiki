# Рабочие процессы (Workflows)

Пошаговые руководства и туториалы для использования в реальных производственных сценариях.

| Рабочий процесс (Workflow) | Описание |
|----------|-------------|
| [**Варианты анимации (Animation Variants)**](animation_variants.md) | Настройка различных анимаций (дублей) для каждого отдельного View Layer. |
| [**Варианты материалов (Material Variants)**](material_variants.md) | Использование системы Variant Switch для выбора конфигурации продукта, цвета и его отделки. |
| [**Пакетный рендеринг (Batch Rendering)**](batch_rendering.md) | Автоматизированный рендеринг всех вариантов и ракурсов за один "присест". |
